# Even or Odd Checker
## Objective

The objective of this assignment is to create a JavaScript program
that checks whether a given number is even or odd.

## Tasks Completed
- Created a JavaScript program to check even or odd numbers
- Used four different variables
- Used if-else statements
- Displayed output using console.log()
- Added comments for better understanding
- Used meaningful variable names

## Files Included
### 1. index.html
Loads the webpage and JavaScript file.

### 2. style.css
Contains simple styling for the webpage.

### 3. script.js
Contains the even or odd checking logic.

## Logic Used
The program uses the modulo (%) operator.
- If `number % 2 === 0`
  → the number is even
- Otherwise
  → the number is odd
